# Tech Stack

## 1. Mục tiêu lựa chọn công nghệ

Tài liệu này đề xuất bộ công nghệ chuẩn cho hệ thống phát triển theo định hướng production-ready, dễ mở rộng, dễ bảo trì, phù hợp cho mô hình web application với:

* Backend sử dụng **Java Spring Boot**
* Frontend sử dụng **Vue.js**
* Kiến trúc ưu tiên **modular, maintainable, secure, scalable**
* Phù hợp cho hệ thống nội bộ doanh nghiệp hoặc SaaS có khả năng mở rộng về sau

---

## 2. Kiến trúc tổng thể đề xuất

Hệ thống được đề xuất theo mô hình:

* **Frontend**: SPA sử dụng Vue 3
* **Backend**: RESTful API sử dụng Spring Boot
* **Database**: PostgreSQL
